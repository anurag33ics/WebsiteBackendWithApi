# Website Backend with API

This project is a backend management system developed in PHP utilizing API functionality. The system is designed for managing administrative features efficiently.

## Features

1. **Admin Management**
   - Add new administrators to manage the backend system.

2. **Query Management**
   - Retrieve and display a list of queries submitted through the website.

3. **Secure Login System**
   - An admin login page to ensure secure access to all features. Accessing features without logging in is restricted.

4. **Admin Dashboard**
   - A comprehensive dashboard displaying the total number of queries.

5. **Export Queries**
   - Manage user queries efficiently with an option to export the list as an Excel or PDF file.

## Technologies Used

- PHP
- API
- HTML/CSS (for the dashboard)
- MySQL (for query and admin data management)

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/anurag33ics/WebsiteBackendWithApi.git
   ```

2. Navigate to the project folder:
   ```bash
   cd WebsiteBackendWithApi
   ```

3. Set up the database:
   - Import the provided SQL file into your MySQL database.
   - Update the database credentials in the configuration file (`config.php`).

4. Start your server:
   - Use tools like XAMPP, WAMP, or any PHP-supported server.

5. Access the admin panel:
   - Navigate to `http://localhost/WebsiteBackendWithApi` in your browser.
